Diseño del sitio web de ARCA
Diseño UX/UI

La totalidad de este proyecto es propiedad exclusiva de Grupo ARCA, por lo tanto, en este portafolio solo se presenta contenido accesible al público, sin revelar detalles relacionados con el proceso de creación.

Descripción del proyecto

Este proyecto se llevó a cabo en colaboración con compañeros con el objetivo de rediseñar el sitio web comercial de Grupo ARCA.

Durante el proceso de diseño se intentó modernizar la interfaz del sitio, además de mostrar de forma legible sus diferentes proyectos.

Buscando simplificar la experiencia de usuario.
